## Q8
Machine Learning refers to the bits that group each other (specifically the 0's and 1's) that the CPU runs.  
Assembly language is basically the same idea, but instead we have the programmer dealing with actual words that map to those bits.

Example using the letter B & 0x2005:

**Assembly:**
```
LDA #$42
STA $2005
```

**Machine Code:**
```
A9 42 8D 05 20
```
